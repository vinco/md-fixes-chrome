## Firefox WebExtension

Requires [web-ext](https://developer.mozilla.org/en-US/Add-ons/WebExtensions/Getting_started_with_web-ext):

~~~sh
$ npm install -g web-ext
~~~

Develop: `$ web-ext run [-f /path/to/your/firefox-bin]`

Build: `$ web-ext build`
